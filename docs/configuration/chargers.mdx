### KEBA Connect

Hardware Preparation

Requires enabled UDP function with DIP switch 1.3 = `ON`, see installation manual.

__NOTE:__ rfid tag and serial tag are optional.
RFID enables EVCC to start charging even if the Wallbox was not unlocked using the chip/card.
`serial` enables EVCC to communicate with the Wallbox when run in Docker. UDP-Port 7090 must be added to the Docker port settings!

### Wallbe Eco/Pro

Hardware preparation

Wallbe chargers are supported out of the box. The Wallbe must be connected using Ethernet. If not configured, the default address `192.168.0.8:502` is used. The default password for the Web-UI is `wallbe7213`.

To allow controlling charge start/stop, the physical configuration must be modified. Once opened, DIP 10 must be set to ON:

  ![DIP Switch image](wallbe_Eco_dip10.jpeg)

More information on interacting with Wallbe chargers can be found at [GoingElectric](https://www.goingelectric.de/forum/viewtopic.php?p=1212583). Use with care.

__NOTE:__ The Wallbe products come in two flavors of the Modbus register layout depeding on the pre-installed firmware. Older models (2017 known to be "old", 2019 known to be "new") do not support some advanced features like current adjustments in 0,1A steps. For such models make sure to set `legacy: true`. You can find you which one you have using [MBMD](https://github.com/evcc-io/evcc/wiki/5):

```
mbmd read -a 192.168.0.8:502 -d 255 -t holding -e int 300 1
```

Compare the value to what you see as Actual Charge Current Setting in the Wallbe web UI. If the numbers match, it's an OLD controller firmware, if the reading is factor 10x the UI value then it's a NEW controller firmware.

__NOTE:__ Opening the wall box must only be done by certified professionals. The box must be disconnected from mains before opening.

### EEBUS (experimental)

- Starte `evcc eebus-cert` (falls noch geschehen)
- Füge die Ausgabe in die `evcc.yaml` Konfigurationsdatei hinzu
- Öffne das Web Oberfläche der Wallbox um die SKI (Identifcation Number) zu finden. Beim Porsche Mobile Charger Connect ist dies unter dem Menü "Connections" und Untermenü "Energy Manager" zu finden.
- Füge den Charger hinzu
  <Config part="chargers" file="eebus" />
- Starte `evcc`
- Auf der Web Oberfläche der Wallbox ist üblicherweise auf der gleichen Seite wie zuvor, sollte nun `EVCC` erscheinen und eine Option um die Wallbox mit `EVCC` zu verbinden (pairen) vorhanden sein. Bitte dies durchführen.
- EVCC sollte nun mit der Wallbox kommunizieren können.
