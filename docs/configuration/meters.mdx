# Hausinstallation

Damit evcc den PV-Überschuss regeln kann, ist in der Konfiguration ein `grid`-Meter erforderlich.
Zusätzlich können noch die Messstellem für Wechselrichter (`pv`) und Hausspeicher (`battery`) hinterlegt werden.

```yaml
meters:
  - name: grid
    type: ...
  - name: pv
    type: ...
  - name: battery
    type: ...
```
